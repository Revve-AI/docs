# VIB Voice Agent — Overview

This document describes the production Voice Agent that VIB uses to call credit-card applicants whose application profiles are incomplete, collect the missing information by phone, and hand the completed record back to VIB's CRM for card issuance.

The agent is implemented on top of Revve Voice with a **Simple Prompt** engine (GPT-4o) and a custom tool pack for phone verification, address normalization, and call termination.

## Agent identity

| Property | Value |

| **Pronunciation** | `VIB` is pronounced `Vi Ai Bi` |
| **Language** | Vietnamese only |
| **Direction** | Outbound (primary) + Inbound (return calls) |
| **Engine** | Simple Prompt |

## Business goal

Complete an applicant's credit-card application profile in a single phone call by:

1. Verifying the caller is the real applicant (phone-number verification against VIB CRM).
2. Collecting the subset of fields still missing on the application (driven by Handlebars conditionals in the prompt).
3. Handing results back to VIB's CRM via webhook.
4. Never selling, negotiating, or improvising beyond the script.
