# Enrolling Contacts

![Campaign enrollments tab showing enrolled contacts and their status](/screenshots/campaign-enrollments.png)

A campaign does not start contacting anyone until leads are **enrolled**. Each enrollment creates a tracked journey for one contact through the campaign's engagement flow.

## Enrollment methods

| Method | Best for |
|--------|----------|
| **CSV Upload** | Bulk enrollment from a spreadsheet or CRM export. |
| **API Integration** | Real-time enrollment from your CRM or internal systems. |
| **Manual enrollment** | Adding individual contacts already in the system. |

## CSV Upload

1. Open the campaign and go to the **Enrollments** tab.
2. Click **Import CSV**.
3. Upload your CSV file.
4. Map the CSV columns to contact fields (phone number, name, email, custom fields).
5. Review the preview and click **Import**.

The system automatically prevents duplicate enrollments — if a contact is already enrolled in this campaign, it will be skipped.

## API Integration

Two endpoints are available for programmatic enrollment:

### Enroll an existing contact

```
POST /api/campaigns/enrollments

{
  "campaignId": "your-campaign-id",
  "contactId": "existing-contact-id"
}
```

### Create a contact and enroll in one call

```
POST /api/campaigns/contact-enrollments

{
  "campaignId": "your-campaign-id",
  "contact": {
    "phone": "+84901234567",
    "firstName": "Nguyen",
    "lastName": "Van A",
    "email": "a@example.com"
  }
}
```

Use this endpoint when the lead does not yet exist in the Revve system. The contact is created and enrolled in a single API call.

## Manual enrollment

1. Open the campaign → **Enrollments** tab.
2. Click **+ Add Enrollment**.
3. Search for an existing contact by name, phone, or email.
4. Select the contact and confirm.

## Enrollment states

Each enrollment moves through these states:

| State | Meaning |
|-------|---------|
| **Active** | The enrollment is live — the AI engine is scheduling and executing actions. |
| **Paused** | Temporarily halted (e.g., prehook failed, manual pause). |
| **Completed** | The lead exited the campaign (success or failure). |
| **Failed** | A system error prevented the enrollment from completing. |

## Tips

- **Start small** — enroll 5–10 test contacts first, verify the campaign works, then enroll larger batches.
- **Check for duplicates** — the system prevents re-enrollment, but verify your CSV doesn't contain contacts already in another active campaign.
- **Monitor after bulk import** — check the campaign dashboard within 30 minutes of a large import to catch issues early.
